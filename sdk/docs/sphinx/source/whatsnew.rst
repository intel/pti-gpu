Version 0.17.0
---------------

*Current development version*

**Updates in this release:**

* Added support for tracing Intel(R) oneCCL library operations (via ``ptiViewEnable(PTI_VIEW_COMMUNICATION)``)
* Added tracing of SYCL Graph operations with capture of repeatedly executed command lists. Improvements of this capability are in progress.
* Added compatibility with Intel oneAPI Compiler 2026.
* Updated to Level Zero Loader version 1.28.0.
